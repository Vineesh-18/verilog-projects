
# Verilog Projects Portfolio

This repository contains my Verilog HDL projects developed as part of my digital design learning and practice. Each project demonstrates different concepts of digital design such as combinational logic, sequential circuits, and finite state machines.

---

## 1. 4-bit ALU (Arithmetic Logic Unit)

### Description
A combinational 4-bit ALU that performs basic arithmetic and logical operations based on a 2-bit select signal.

### Operations
- AND
- OR
- ADD
- SUBTRACT

### Files
- alu_4bit.v
- tb_alu.v
- outputALU.txt

---

## 2. MOD-10 Counter with 7-Segment Display

### Description
A synchronous MOD-10 counter that counts from 0 to 9 and resets back to 0. The output is displayed using a 7-segment display decoder.

### Features
- Synchronous MOD-10 counter
- Clock divider for slow clock generation
- 7-segment display decoder
- Testbench for simulation

### Files
- mod_10_counter.v
- seven_segment.v
- clock_divider.v
- mod_10_display.v
- tb_mod10.v
- output.txt

---

## 3. Traffic Light Controller

### Description
A traffic light controller designed using a Finite State Machine (FSM). It controls signals for two roads and cycles through different states. A clock divider is used for simulation visibility.

### Features
- FSM-based design
- Two-road traffic signal control
- Clock divider for timing control
- Testbench for verification

### Files
- traffic_light_controller.v
- clock_divider.v
- tb.v
- output
